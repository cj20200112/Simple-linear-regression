# 介紹
本專案使用salary.csv檔案之內容，透過簡單回歸依據年資來預測薪資，同時也使用梯度下降法來找出更準確的最佳解

# 使用套件
pandas、numpy、scipy.stats、wget、matplotlib.font_manager、ipywidgets.interact、matplotlib、matplotlib.pyplot
